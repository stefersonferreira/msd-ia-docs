# Integrando o Fluxo Operacional

**Estudo de Caso:** Sistema SMNC

Nesta etapa, integramos o sistema ao fluxo operacional existente, garantindo que a solução se encaixe naturalmente no dia a dia da fábrica.

## 1. Mapeamento do Fluxo Operacional

Entender o fluxo operacional atual é essencial para garantir que o sistema seja adotado pelos usuários.

::: info OBJETIVO
Mapear o fluxo operacional atual e definir como o sistema se integra a ele, minimizando mudanças disruptivas.
:::

## 2. O Prompt de Integração

```md
Com base na especificação do SMNC, elabore um mapeamento do fluxo operacional:

1. **Fluxo Atual (As-Is)**: Documente o fluxo atual de identificação e tratamento de não conformidades.
   - Quais são os passos?
   - Quem participa de cada etapa?
   - Quais são os pontos de dor no fluxo atual?

2. **Fluxo Proposto (To-Be)**: Defina como o fluxo ficará com o sistema.
   - Onde o sistema se integra?
   - Quais etapas são automatizadas?
   - Quais etapas permanecem manuais?

3. **Pontos de Integração**: Identifique onde o sistema precisa se integrar com outros sistemas/processos.
   - Integração com ERP
   - Integração com processos de qualidade
   - Integração com relatórios gerenciais

4. **Plano de Adoção**: Defina como os usuários serão treinados e como a mudança será comunicada.
```

::: tip NOTA DO SÊNIOR
Um bom mapeamento de fluxo operacional garante que o sistema seja percebido como uma melhoria, não como um obstáculo, aumentando a taxa de adoção.
:::

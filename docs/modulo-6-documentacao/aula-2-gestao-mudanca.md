# Gestão de Mudança e Implantação

Software não roda no vácuo; roda em empresas com pessoas reais. Vamos definir como implantar o SMNC sem parar a fábrica.

## Prompt de Fases de Implantação

```md
Considerando essa especificação, é viável criar um modo de transição onde o sistema aceite inputs manuais sem validação rígida nas primeiras semanas?

Defina 3 fases de implantação:
1. **Shadow Mode** (Monitoramento sem bloqueio do lote).
2. **Hybrid Mode** (Bloqueio apenas em defeitos críticos).
3. **Full Governance** (Bloqueio total conforme regras).

Detalhe as configurações necessárias no software para cada fase.
```
